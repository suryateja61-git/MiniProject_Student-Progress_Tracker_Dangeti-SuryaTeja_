
# 📚 Student Progress Tracker System

A simple Python-based Student Progress Tracker that allows users to enter student details, record marks for multiple subjects, calculate total and average marks, assign grades, and generate a progress report.

## 🚀 Features

* Enter student details:

  * Name
  * Class
  * Roll Number
* Record marks for multiple subjects
* Calculate:

  * Total Marks
  * Average Marks
* Generate grades based on average score
* Display Pass/Fail status
* Menu-driven interface
* View reports multiple times

## 🛠️ Technologies Used

* Python 3

## 📋 Grade Criteria

| Average Marks | Grade |
| ------------- | ----- |
| Above 98      | O     |
| Above 90      | A     |
| 76 - 89       | B     |
| 51 - 74       | C     |
| 36 - 49       | D     |
| 35 and Below  | F     |

## ▶️ How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd student-progress-tracker
```

3. Run the program:

```bash
python student_tracker.py
```
